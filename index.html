Aujourd'hui,je vous montre comment crée une page web grâce au language html
NB:Ceci n'est une chose facile

Le HTML (HyperText Markup Language) est un langage de balisage utilisé pour crée et structurer les page web.

Il permet d'organiser le contenu d'une page sur Internet,comme:

  • les titres
  • les pargraphes
  • les images
  • les liens
  • les tableaux
